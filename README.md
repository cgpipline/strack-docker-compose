# Linux 环境一键安装脚本

基于 docker-compose 环境，需要大家自己安装好 docker 和 docker-compose 环境

# 命令

下载本仓库代码到自己服务器，进入strack-install根目录

|命令|说明|
|---|---|
|./run.sh install| 全新安装strack，一键安装过了会卸载重新安装|
|./run.sh update| 同步strack最新仓库代码，更新strack|
|./run.sh uninstall| 卸载strack|
